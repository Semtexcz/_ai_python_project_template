# Conversation Summaries

This directory stores concise summaries of important AI-assisted project conversations.

Rules:

- Do not store raw full conversations by default.
- Store only curated summaries.
- Important decisions must be copied or linked into `docs/decisions/`.
- Current facts must be copied into `docs/CURRENT_STATE.md`.
- Concrete work must become tasks in `project/backlog/`.
- Risks must be copied into `docs/RISKS.md`.
- Conversation summaries are archival notes, not the main source of truth.

Use this naming convention:

```text
YYYY-MM-DD-short-topic.md
```

Example:

```text
2026-06-11-auth-and-long-running-jobs.md
```
